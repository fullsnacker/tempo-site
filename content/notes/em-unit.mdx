Em depends on the font-size of the parent, so we can use EM in paddings, gaps o margins relatives to the font-size of the element

<CH.Code>

```css
.btn {
	font-size: 1.5rem;
	padding: 1em;
	margin: 1em;
}
```

</CH.Code>
