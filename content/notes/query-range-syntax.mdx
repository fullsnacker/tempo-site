Now we have in CSS new media queries syntax with the logical operator of javascript:

<CH.Code>

```css
@media (width >= 600px) {
	/*...styles*/
}

/* When the browser is between 400px - 1000px  */
@media (400px <= width <= 1000px) {
	/*...styles*/
}
```

</CH.Code>
